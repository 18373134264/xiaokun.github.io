# 1.jenkins 下载

因为我仍然使用的是 jdk1.8 进行开发，所以我下载的是 jenkins2.332.1 版本（jenkins2.346.1 版本在 2022 年末不再支持 java8，如果项目使用的是 jdk11 可以继续使用该 jenkins 版本），更多版本下载请点击 jenkins 下载

# 2.jenkins 安装

找到下载的 jenkins.war 包可以放到自己指定的路径，如图我放到的位置是 D:\ 下，然后再在该文件目录输入 cmd 进入命令行，使用 java -jar jenkins.war 命令运行 jenkins

jenkis 密码位置：C:\Users\3hyxk.jenkins\secrets\initialAdminPassword
输入上面返回的密码 61fc6441ec8e490b8d4e403a3d828557
安装插件直接选择第一个
之后访问 localhost:8080 输入账号密码就能正常使用了

# 3.jenkins 配置

  • jenkins 设置中文方法 (如果失败请自寻方法)
  1. 进入 Manage Jenkins 中，点击 Manage Plugins。
  2. 安装插件 Locale plugin
  3. 在 Locale 配置下 Default Language 设定 zh_CN 简体中文后重启 jenkins


4. 在对应的地方输入代码 git 地址及账号密码，之后指定分支
